# c-cli-make-cmake-multi-file-hello-world

## Description
This is a POC project to demonstrate 
make with cmake as a build tool.

## Tech stack
- c
- make
- cmake

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
